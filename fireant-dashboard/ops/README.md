# 📂 ops/ — 운영/배포 디렉토리

이 디렉토리는 **운영, 배포, 자동화**와 관련된 스크립트 및 데이터 파일들이 위치합니다.

## 주요 파일 유형

| 유형 | 설명 |
|------|------|
| `*.py` | 백테스트, ETF 시뮬레이션, 데이터 수집 등 Python 스크립트 |
| `*.js` | 폼 생성, 이미지 처리 등 Node.js 스크립트 |
| `*.sh` | ETF 캡처 등 쉘 스크립트 |
| `*.json` | 피드, 이벤트, 프리미엄 데이터 등 운영 데이터 |
| `*.csv` | 분석 결과 데이터 |
| `*.txt` | 백테스트 결과 리포트 |

## 주요 기능

- **백테스트**: 과거 데이터 기반 전략 검증 (`backtest_*.py`)
- **ETF 캡처**: 자동 ETF 차트 캡처 (`etf_capture.sh`, `etf_image_gen.py`)
- **봇 운영**: 텔레그램 봇 관련 스크립트 (`fireantagent_bot.py`)
- **데이터 관리**: 피드/이벤트 JSON 생성 및 업데이트

## 주의사항
