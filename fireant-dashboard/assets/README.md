# 📂 assets/ — 정적 에셋 디렉토리

이 디렉토리는 **이미지, 아이콘, 배너** 등의 정적 에셋 파일들이 위치합니다.

## 파일 유형

| 유형 | 설명 |
|------|------|
| `*.png` | 배너, 프로필 이미지, 이벤트 포스터 등 |
| `*.jpg` | 사진 및 포맷 이미지 |
| `*.svg` | 벡터 아이콘 (필요시 추가) |

## 주요 에셋

- **이벤트 배너**: 밸리언즈 등 이벤트용 배너 이미지
- **네이버 배너**: 네이버 카페/블로그용 프로필 및 배너
- **AI 에이전트 배너**: AI 기능 홍보용 배너 이미지
- **브랜드 에셋**: FireAnt 로고 및 브랜딩 이미지

## 주의사항

- 대용량 파일은 Git LFS 사용을 고려하세요.
- 파일명은 `YYYY-MM-DD-설명` 형식을 권장합니다.
