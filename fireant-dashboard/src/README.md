# 📂 src/ — 소스 코드 디렉토리

이 디렉토리는 **FireAnt Dashboard**의 핵심 소스 코드와 HTML 페이지들이 위치합니다.

## 하위 구성

| 경로 | 설명 |
|------|------|
| `bots/` | 봇 관련 소스 코드 |
| `events/` | 이벤트 페이지 소스 |
| `feed/` | 피드 관련 소스 코드 |
| `indicators/` | 지표 계산 및 시각화 소스 |
| `kr-exchange/` | 한국 거래소 관련 소스 |
| `leaderboard/` | 리더보드 관련 소스 |
| `winners/` | 승자(우수 트레이더) 관련 소스 |
| `fireant-todo/` | 내부 TODO 관리 소스 |
| `*.html` | 개별 대시보드 페이지 (BTC 상관분석, ETF 차트, 이벤트, 지표 등) |

## 주의사항

- 이 디렉토리의 HTML 파일은 상대경로를 사용합니다.
- 파일 이동 시 `index.html`과의 경로 관계를 반드시 확인하세요.
