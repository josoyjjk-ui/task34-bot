# Self-Learning Skill

딸수(메인 에이전트)의 자기 개선 시스템.

## 동작 방식

1. **매일 새벽 3시** — 최근 세션/메모리 로그 분석 → 실수 패턴 감지
2. **memory/lessons.md** — 패턴 + 규칙 후보 누적 저장
3. **매주 월요일 9시** — 주간 학습 리포트를 해병님에게 보고
4. **해병님 승인** — "반영해" 한 마디로 AGENTS.md 자동 업데이트

## 감지 패턴

- 완료보고 누락 (사용자가 먼저 상태를 물어봄)
- 같은 작업 반복 재요청 (3회 이상)
- 품질 불만 (다시 해, 깨졌다 등)
- 타임아웃 반복 (같은 작업에서 2회 이상 타임아웃)

## 파일

- `scripts/analyze_session.py` — 분석 스크립트
- `memory/lessons.md` — 감지된 패턴 누적
